## How to Run
1. Clone: `git clone https://github.com/altinhamzaa/EmployeeAPI.git`
2. Navigate: `cd EmployeeAPI`
3. Restore: `dotnet restore`
4. Build: `dotnet build`
5. Run: `dotnet run`

## Testing
- API: http://localhost:5083/api/employees
- Swagger: http://localhost:5083/swagger

## Features
- ASP.NET Core 8 Web API
- Entity Framework Core + InMemory DB
- CRUD operations for Employee entities
- Age calculation from date of birth
- Data validation
